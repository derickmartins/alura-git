Lista de Alguns Codigos do Git

  • cd: selecionar o documento.
  • ls: selecionar o arquivo.
  • git init: cria/inicializa um repositorio.
  • git status: vizualizar informações do repositorio.
  • git add: adicona um arquivo a ser monitorado.
  • git pull: pucha os dados de um locar para outro.
  • git push: envia os dados de um local para outro.
